# 🔧 ПРОВЕРКА ИСТОРИЙ И ФУНКЦИЙ

## ✅ КАК ПРОВЕРИТЬ ЧТО ВСЁ РАБОТАЕТ

### 1. Откройте консоль (F12)
Должны быть сообщения:
```
[Stories] Модуль загружен, инициализация...
[Stories] Вызываем loadStoriesForSidebar
[Stories] Все функции экспортированы
```

### 2. Проверьте функции в консоли:
```javascript
// Вставьте в консоль:
console.log('=== ПРОВЕРКА ИСТОРИЙ ===');
console.log('showCreateStoryModal:', typeof window.showCreateStoryModal);
console.log('viewUserStories:', typeof window.viewUserStories);
console.log('loadStoriesForSidebar:', typeof window.loadStoriesForSidebar);
console.log('===================');

// Проверка блока историй
const container = document.getElementById('storiesContainer');
const list = document.getElementById('storiesList');
console.log('storiesContainer:', container ? 'НАЙДЕН' : 'НЕ НАЙДЕН');
console.log('storiesList:', list ? 'НАЙДЕН' : 'НЕ НАЙДЕН');

// Принудительная загрузка
if (typeof loadStoriesForSidebar === 'function') {
    loadStoriesForSidebar();
    console.log('✅ Истории загружены');
} else {
    console.log('❌ Функция не найдена');
}
```

### 3. Где должен быть блок историй:
```
┌─────────────────────────────┐
│  Сайдбар                    │
├─────────────────────────────┤
│  [Мой аватар] [Имя]         │
│  В сети                     │
├─────────────────────────────┤
│  🔍 [Поиск]                 │
├─────────────────────────────┤
│  [ИСТОРИИ - НОВЫЙ БЛОК]     │  ← ВОТ ЗДЕСЬ!
│  [+] [Друг1] [Друг2] ...    │
├─────────────────────────────┤
│  [Все] [Личные] [Группы]    │
│  [Непрочитанные]            │
├─────────────────────────────┤
│  [Список чатов]             │
│  ...                        │
└─────────────────────────────┘
```

### 4. Как создать историю:
1. Нажмите **[+]** в блоке историй (сверху сайдбара)
2. Выберите фото или сделайте фото
3. Добавьте подпись
4. Нажмите "Опубликовать"

### 5. Как посмотреть историю:
1. Нажмите на аватар друга в блоке историй
2. История откроется на весь экран
3. Переключение: стрелки ◀ ▶ или тап по краям

## 🐛 ЕСЛИ ИСТОРИЙ НЕТ

### Проверьте:
1. **Блок историй виден?**
   - Должен быть сверху сайдбара
   - Серый фон
   - Кнопка [+] и аватары

2. **Функции работают?**
   ```javascript
   // В консоли:
   showCreateStoryModal()  // Должно открыть модалку
   ```

3. **Firebase подключен?**
   ```javascript
   // В консоли:
   db.ref('storiesIndex').once('value').then(s => console.log(s.val()))
   ```

### Если не работает:
1. Обновите страницу (Ctrl+F5)
2. Проверьте консоль на ошибки
3. Убедитесь что все JS файлы загружены
