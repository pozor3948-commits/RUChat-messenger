## [1.6.0] - 14 марта 2026

### 🔒 Безопасность
- **ДОБАВЛЕНО**: PBKDF2 хеширование паролей (SHA-256, 100,000 итераций, соль 16 байт)
- **ДОБАВЛЕНО**: Обратная совместимость со старыми хешами паролей
- **ДОБАВЛЕНО**: Функции `safeHtml()` и `escapeHtmlAttr()` для XSS защиты
- **ИСПРАВЛЕНО**: Удален хардкод токенов из `bot.js`
- **ИСПРАВЛЕНО**: Унифицированы Firebase проекты (бот и приложение)
- **ИСПРАВЛЕНО**: TURN серверы вынесены в конфигурацию
- **ДОБАВЛЕНО**: Валидация токена бота при запуске

### ⚡ Производительность
- **ДОБАВЛЕНО**: Debounce для поиска контактов (300мс)
- **ДОБАВЛЕНО**: Debounce для поиска сообщений (300мс)
- **ИСПРАВЛЕНО**: Утечки памяти в WebRTC звонках
- **ДОБАВЛЕНО**: Очистка всех обработчиков PeerConnection
- **УЛУЧШЕНО**: Обработка ошибок в Cloud Functions

### 🆕 Новые функции
- **ДОБАВЛЕНО**: Резервное копирование данных (`js/backup.js`)
  - Создание резервной копии аккаунта и чатов
  - Восстановление из резервной копии
  - Автоматическое копирование раз в 7 дней
  - Скачивание файла backup
  - Локальное хранение в localStorage
- **ДОБАВЛЕНО**: PWA manifest включен

### 📝 Документация
- **ДОБАВЛЕНО**: `.env.example` - шаблон конфигурации
- **ДОБАВЛЕНО**: `SECURITY_REPORT.md` - отчет о безопасности
- **ДОБАВЛЕНО**: `SETUP_GUIDE.md` - пошаговая инструкция настройки
- **ДОБАВЛЕНО**: `FINAL_REPORT.md` - финальный отчет
- **ОБНОВЛЕНО**: `README.md` - полная переработка
- **ОБНОВЛЕНО**: `database.rules.json` - новые правила безопасности

### 🐛 Исправления
- **ИСПРАВЛЕНО**: Обработка ошибок в `sendPushToUser()` (Cloud Functions)
- **ИСПРАВЛЕНО**: Завершение звонка при разрыве соединения
- **ИСПРАВЛЕНО**: Очистка ICE кандидатов после звонка
- **ИСПРАВЛЕНО**: Валидация паролей в `recoverPassword()`
