- Добавлена `escapeHtmlAttr()` для атрибутов

**Код:**
```javascript
function safeHtml(strings, ...values) {
    return strings.reduce((result, string, i) => {
        const escaped = escapeHtml(String(values[i] ?? ''));
        return result + string + escaped;
    }, '');
}
```

## ⚠️ ИЗВЕСТНЫЕ ОГРАНИЧЕНИЯ

### 1. Firebase Authentication
**Проблема:** Требуется ручная настройка в Firebase Console

**Решение:** Следуйте инструкции в `SETUP_GUIDE.md`

---

### 2. Обратная совместимость паролей
**Проблема:** Старые пароли хешировались простой функцией

**Решение:** 
- Реализована обратная совместимость
- При следующем входе пароль будет захеширован по-новому
- Администратор может сбросить пароль через бота

---

### 3. PWA через file://
**Проблема:** PWA не работает через file://

**Решение:** Используйте `firebase serve` или другой HTTP сервер
