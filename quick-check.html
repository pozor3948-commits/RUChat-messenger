🔍 RuChat - Проверка

Откройте index.html и консоль (F12)

Инструкция:

  1. Откройте index.html
  2. Нажмите F12 (консоль)
  3. Вставьте этот код:
// Проверка функций:
console.log('=== ФУНКЦИИ ===');
['showCreateStoryModal','viewUserStories','loadStoriesForSidebar','sendLocation','showAlbumCreator','showCreatePollModal'].forEach(fn => {
    console.log((typeof window[fn]==='function'?'✅':'❌') + ' ' + fn);
});

// Проверка элементов:
console.log('=== DOM ===');
['storiesContainer','storiesList','chatFoldersContainer','friendList','mobileChatActionsBar'].forEach(id => {
    console.log((document.getElementById(id)?'✅':'❌') + ' #' + id);
});

// Загрузить истории:
if (typeof loadStoriesForSidebar==='function') loadStoriesForSidebar();

Если функции не найдены:

1. Обновите страницу (Ctrl+F5)

2. Проверьте что все скрипты загружены в index.html

3. Посмотрите ошибки в консоли
