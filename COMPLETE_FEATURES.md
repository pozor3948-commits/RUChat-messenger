## 🔐 E2EE ШИФРОВАНИЕ - ПОДРОБНО

### Алгоритм:
- **RSA-OAEP** с длиной ключа 2048 бит
- **SHA-256** хеш
- Ключи генерируются на устройстве
- Приватный ключ не передается

### Как работает:
1. Генерация пары ключей (публичный/приватный)
2. Публичный ключ сохраняется в базе
3. При отправке сообщение шифруется публичным ключом
4. Расшифровать можно только приватным ключом

### Использование:
1. Нажмите **🔒** в сайдбаре
2. "Новый секретный чат"
3. Введите имя пользователя
4. Сообщения шифруются!
