# 🍬 КОНФЕТКА v1.6.2c

## ✅ ЧТО ИСПРАВЛЕНО

### 1. Кэш браузера
**Проблема:** Браузер загружал старые версии файлов  
**Решение:**
- Добавлены meta-теги для запрета кэширования
- Добавлен скрипт очистки кэша при загрузке
- Все файлы используют версию `v=1.6.2c`

---

### 2. openMediaViewer is not defined
**Проблема:** Функция называлась `openMedia` а экспортировалась как `openMediaViewer`  
**Решение:** Удалён лишний экспорт, оставлен только `window.openMedia`

---

### 3. Service Worker 404 ошибка
**Проблема:** Файл `sw.js` отсутствует  
**Решение:** Отключена регистрация Service Worker

---

### 4. Backup Permission Denied
**Проблема:** Нет прав на запись в Firebase  
**Решение:**
- Обёрнуто в try-catch
- Отключено автоматическое резервное копирование
- Сохранение только в localStorage

---

### 5. WebRTC звонки
**Улучшения:**
- 6 STUN серверов (было 2)
- 4 TURN сервера (было 3)
- Обработка ICE ошибок (не спамит)
- Unified Plan SDP

---

## 🚀 КАК ЗАПУСТИТЬ

### 1. Очисти кэш (ОДИН РАЗ)
Открой в браузере:
```
clear-cache.html
```
Или вручную: `Ctrl+Shift+Delete` → Clear cache

### 2. Открой приложение
```
index.html
```

### 3. Проверь консоль (F12)
Должно быть:
```
[RuChat] Версия: 1.6.2
[Cache] Удалён кэш: ...
Build 2026-03-14c v1.6.2 (CONFETKA)
```
