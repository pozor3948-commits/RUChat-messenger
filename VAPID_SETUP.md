# 🔔 Настройка Push-уведомлений (VAPID Key)

## ✅ Статус
**VAPID ключ настроен!** Push-уведомления готовы к работе.

## Проверка

1. Откройте мессенджер в браузере
2. Разрешите уведомления, когда браузер запросит
3. Отправьте сообщение с другого устройства/аккаунта
4. Вы должны получить push-уведомление

## Если уведомления не приходят

1. Проверьте разрешение браузера на уведомления
2. Убедитесь, что вкладка не активна (push приходит только когда чат не открыт)
3. Проверьте консоль браузера на ошибки

## Примечания

- VAPID ключ **публичный** — безопасно хранить в клиентском коде
- Push работает только через **HTTPS** или **localhost**
- Уведомления приходят, только когда вкладка не активна
